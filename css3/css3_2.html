用 CSS3，你可以创建圆角边框，添加阴影框，并作为边界的形象而不使用设计程序

border-radius 圆角

border-radius 属性允许您为元素添加圆角边框！
div {
border:2px solid;
border-radius:25px;
}

box-shadow盒阴影

box-shadow 属性被用来添加阴影:
div {
box-shadow: 10px 10px 5px #888888;
}

border-image边界图片

border-image 属性允许你指定一个图片作为边框！
这里，图像平铺（重复）来填充该区域。
border-image:url(border.png) 30 30 round;
-webkit-border-image:url(border.png) 30 30 round; /* Safari 5 and older */
-o-border-image:url(border.png) 30 30 round; /* Opera */

新边框属性

css3属性
说明
border-image
设置所有边框图像的速记属性
border-radius
一个用于设置所有四个边框- *-半径属性的速记属性
box-shadow
附加一个或多个下拉框的阴影
