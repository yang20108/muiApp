HTML 区块元素

大多数 HTML 元素被定义为块级元素或内联元素，块级元素在浏览器显示时， 通常会以新行来开始（和结束）。比如：<h1>,<p>,<ul>,<table>

HTML 内联元素

内联元素在显示时通常不会以新行开始。比如：<b>,<td>,<a>,<img>

HTML <div>元素

<div> 元素是块级元素，它可用于组合其他 HTML 元素的容器。

HTML <span>元素

<span> 元素是内联元素，可用作文本的容器。也没有特定的含义。当与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。

HTML 布局

div 元素是用于分组 HTML 元素的块级元素。
