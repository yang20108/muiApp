导航栏 - 垂直

  • 主页
  • 联系
  • 关于
  • 我的
导航条基本上是一个链接列表，所以使用 <ul> 和 <li>元素非常有意义：
<ul>
<li><a href="#">主页</a></li>
<li><a href="#">联系</a></li>
<li><a href="#">关于</a></li>
<li><a href="#">我的</a></li>
</ul>
现在，让我们从列表中删除边距和填充：
ul {
list-style-type: none;
margin: 0;
padding: 0;
width: 200px;
background-color: #f1f1f1;
}
垂直导航栏:
li a {
display:block;
color: #000000;
padding: 8px 16px;
text-decoration: none;
}
在鼠标移动到选项时，修改背景颜色：
li a:hover {
background-color: #555;
color: white;
}
