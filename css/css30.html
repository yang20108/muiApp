具有特定属性的HTML元素样式不仅仅是class和id 。

属性选择器

[title] {
color: blue;
}

<h2 title="h2"></h2>

属性和值选择器

[title=value] {
border: 2px solid salmon;
}
<div title="value">title="value"</div>
<div title="val">title="val"</div>

属性和值的选择器 - 多值

[title~=hello] {
color: seashell;
}
<div title="ting hello">title="ting hello"</div>
<div title="hello world">title="hello world"</div>
<div title="chen">title="chen"</div>
[lang|=en] {
color: aqua;
}
<div lang="en">Hello!</div>
<div lang="en-us">Hi!</div>
<div lang="en-gb">Ello!</div>
<div lang="us">Ello!</div>
比较符号讲解：~= 、*= 、|= 、^= 、$= 的区别
attribute 属性中包含 value:
[title~=flower] <img title="tulip flower"/>
[title*=flower] <img title="ffffflowerrrrrr"/>
attribute 属性以 value 开头:
[lang|=en] <p lang="en-us"/>
[lang^=en] <p lang="ennn"/>
attribute 属性以 value 结尾:
[attribute$=value] <a[src$=".pdf"]/>
