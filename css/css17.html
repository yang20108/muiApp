为了尽量减少样式表中有很多具有相同样式的元素代码，你可以使用分组选择器。每个选择器用逗号分隔。
h1, h2, p {
color: green;
}

嵌套选择器

它可能适用于选择器内部的选择器的样式。
下面的例子设置了三个样式：
p {
color: blue;
text-align: center;
}
讲解：p { }为所有p元素指定一个样式。
.marked {
background-color: red;
}
讲解：.marked { }为所有class="marked"的元素指定一个样式。
.marked p {
color: white;
}
讲解：.marked p { }为所有class="marked"元素内的p元素指定一个样式。
p.marked {
text-decoration: underline;
}
讲解：p.marked { }为所有class="marked"的p元素指定一个样式。
