表格边框

指定CSS表格边框，使用border属性。
FirstnameLastname
PeterGriffin
PeterGriffin
table,th,td {
border:1px solid black;
}
请注意，在上面的例子中的表格有双边框。这是因为表和th/ td元素有独立的边界。

折叠边框

border-collapse 属性设置表格的边框是否被折叠成一个单一的边框或隔开：
FirstnameLastname
PeterGriffin
PeterGriffin
table {
border-collapse: collapse;
}
