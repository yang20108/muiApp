position 属性指定了元素的定位类型
  • static
  • relative
  • fixed
  • absolute
  • sticky
元素可以使用的顶部，底部，左侧和右侧属性定位。然而，这些属性无法工作，除非是先设定position属性。 他们也有不同的工作方式，这取决于定位方法

static 定位

静态定位的元素不会受到 top, bottom, left, right影响
该元素使用了 position: static;

relative 定位

相对定位元素的定位是相对其正常位置, 会受到 top, bottom, left, right影响
该元素使用了 position: relative;
该元素使用了 position: relative; 并配合left使用
该元素使用了 position: relative; 并配合right使用

fixed 定位

元素的位置相对于浏览器窗口是固定位置。即使窗口是滚动的它也不会移动 , 会受到 top, bottom, left, right影响
fixed 顶部右侧定位
fixed 该元素在顶部右侧定位

absolute 定位

绝对定位的元素的位置相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于<html> , 会受到 top, bottom, left, right影响
该元素使用了 position: absolute;
定位在父元素的右下角位置
absolute 底部右侧定位

sticky 定位

sticky 英文字面意思是粘，粘贴，所以可以把它称之为粘性定位。
position: sticky;基于用户的滚动位置来定位。
