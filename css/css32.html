使用计数器自动编号

CSS 计数器根据规则来递增变量，使用到以下几个属性:
  • counter-reset - 创建或者重置计数器
  • counter-increment - 递增变量
  • content - 插入生成的内容
  • counter() 或 counters() 函数 - 将计数器的值添加到元素
